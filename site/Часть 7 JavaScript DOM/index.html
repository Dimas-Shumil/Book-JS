Большая часть работы с веб-приложениями, которые связаны с JavaScript, — это обработка событий на странице и выполнение действий, которые должны происходить при обработке этих событий.

В этом модуле подробно поговорим о работе в браузере. А именно:

• Об основных инструментах, которыми удобно пользоваться при разработке;

• Как работать со страницей и взаимодействовать с пользователем посредством подписки на события.

Это ключевые знания, с которыми нужно разобраться для качественной веб-разработки.

Кроме этого, в Тренажёре по JavaScript вы можете дополнительно отработать эту тему в разделах:

1. DOM (3 задания);

2. Событийно-ориентированное программирование (4 задания);

3. Регулярные выражения (5 заданий).
