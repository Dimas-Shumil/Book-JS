Для взаимодействия с пользователем в JavaScript имеется несколько встроенных функций:

➠prompt() — даёт возможность задать вопрос пользователю и ввести ему ответ в текстовое поле.

➠alert() — о чём-нибудь информирует пользователя.

➠confirm() — даёт возможность пользователю подтвердить выбор или отказаться от него.

Все эти функции вызывают всплывающее окно. Пока оно отображается, работа с сайтом полностью блокируется.

Рассмотрим работу с данными функциями подробнее.

prompt

Чтобы вызвать функцию достаточно написать prompt(). Обязательно в конце нужно поставить круглые скобки. Внутри скобок может находиться любой текст. Данный текст увидит пользователь, когда будет вызвана функция.

После вызова функции, пользователь может ввести любое значение в текстовое поле, находящееся внутри всплывающего окна.

Мы можем присвоить данную функцию переменной. Значение из поля ввода будет записано как значение переменной:

const promptText = prompt('Какой-то текст');

console.log(promptText); // “Текст из поля для ввода”

Значение из поля ввода имеет тип string.

alert

Это информационное всплывающее окно.

После отправки формы, его можно показать с текстом «Ваша заявка принята».

Или показывать его при запрете каких-либо действий — «Данная функция вам недоступна».

confirm

Данная функция спрашивает пользователя уверен ли он в своих действиях.

Она может использоваться при удалении элемента таблицы или списка, перехода на следующий шаг формы.

Также как и prompt() мы можем присвоить данную функцию переменной.

Если пользователь нажмёт «OK», то значением переменной будет true.

При нажатии «Отмена», значением переменной будет false.

const really = confirm("вы точно хотите выйти");

console.log(really);

Для решения следующих заданий вы можете воспользоваться сервисом-песочницей.

Задание 7.1

Запросите у пользователя его имя с помощью prompt(). Запишите его в переменную и выведите с помощью alert фразу «Привет, {имя пользователя}!»

Входные данные: Августин

Выходные данные: Привет, Августин!

const userName = prompt('Введите своё имя');
alert(`Привет, ${userName}!`);

Задание 7.2

У нас есть сервис по продаже чего угодно. Попросите пользователя ввести название вещи, которую он хочет продать. Затем попросите ввести цену за которую он готов расстаться с вещью. После ввода цены пользователь должен подтвердить данную цену.

Входные данные: Шкаф, 3000

Выходные данные: true.

const thing = prompt('Что вы хотите продать?');
const thingPrice = prompt(`За сколько вы хотите продать ${thing}?`);
const assurance = confirm(`Вы уверены, что готовы продать ${thing} за ${thingPrice}?`);
