Массив (array) — это последовательное расположение элементов данных в сочетании с индексом элемента данных. Массив имеет фиксированную размерность.

Другими словами, это особая структура данных для хранения упорядоченных коллекций.

Однако, в отличие от Python, массивы в JavaScript могут состоять из данных различного типа. Это значит, что может быть массив, состоящий только из чисел — [1, 2, 3], а может быть массив, состоящий из абсолютно всех типов данных в разных комбинациях — [true, 'string', 123, false, null].

В JavaScript массивы являются подмножеством объектов, то есть любой массив — это объект. На практике часто возникает потребность узнать, является ли некоторая переменная массивом.

Мы могли бы использовать оператор typeof, о котором говорили ранее, но с массивами он работает не совсем так, как ожидается:

typeof {};

"object"

typeof [];

"object"

Поэтому для того, чтобы узнать, является ли переменная массивом, можно воспользоваться следующими проверками:

Наличие свойства length

const obj = {a: 1, b: 2};

console.log(obj.length);

undefined

const arr = [1, 2];

console.log(arr.length);

2

Свойство length — встроенное. Оно возвращает количество элементов массива. В большинстве случаев типа object и наличия свойства length достаточно, чтобы считать элемент массивом.

Метод Array.isArray()

const obj = {a: 1, b: 2};

console.log(Array.isArray(obj)) ;

false

const arr = [1, 2];

console.log(Array.isArray()arr)

True

Методы работы с массивами

Над массивами можно выполнять множество операций: добавлять и удалять элементы, изменять, перебирать элементы, копировать. Традиционно методы делят на мутирующие и немутирующие.

Мутирующие методы изменяют исходный массив, а немутирующие — не изменяют.

Мутирующие методы:

.push(...items), .unshift(...items) Добавляют элементы в конец/начало массива.
.pop(), .shift() Удаляют элемент из конца/начала массива.

Пример:

let arr = [ "orange"];

arr.unshift("apple");// arr = ["apple", "orange"];

arr.push("strawberry");// arr = ["apple", "orange", "strawberry"];

arr.shift(); //arr = ["orange", "strawberry"];

arr.pop();// arr = ["orange"];

Как следует из примеров, с помощью методов push и pop мы работаем как раз с хвостом массива: добавляется и удаляется элемент с индексом length - 1 (вычитать единицу нужно, так как свойство length возвращает количество элементов, а индексы нумеруются с 0).

Если же необходимо добавить или удалить элемент в начале массива, нам пригодятся методы shift и unshift.

.splice(index[, deleteCount, elem1, ..., elemN]) Удаляет и добавляет элементы.

let arr = ["Я", "прохожу", "курс"];

// с конца массива (берем длину массива)

// удалить 0 элементов

// вставить элементы "в", "Skillfactory"

arr.splice(arr.length, 0, "в", "Skillfactory");

Результат:

let arr = ["Я", "прохожу", "курс"];

arr.splice(arr.length, 0, "в", "Skillfactory");

console.log( arr );

["Я", "прохожу", "курс", "в", "Skillfactory"]

.concat() Склеивает массивы.

Пример:

let arr = [ "Сегодня"];

arr.concat(["солнечно"]);

["Сегодня", "солнечно"]

Немутирующие методы:

.map(cb)

Перебирающий метод массива, принимающий как аргумент функцию cb.

Он вызывает функцию для каждого элемента массива и возвращает массив результатов выполнения этой функции.

Синтаксис:

let result = arr.map(function(item, index, array) {

// возвращается новое значение вместо элемента

});

Где:

item — очередной элемент массива,
index — его индекс,
array — сам массив.

Пример:

let arr = [1,2,3];

let result = arr.map(function(item, index, array) {

return item + 1

});

console.log(arr);

[1, 2, 3]

console.log(result);

[2, 3, 4]

.reduce(cb) или reduceRight(cb) Эти методы используются для вычисления какого-нибудь единого значения на основе всего массива.

Синтаксис:

let result = arr.reduce(function(previousValue, item, index, array) {

// ...

}, [initial]);

Где:

item — очередной элемент массива,
index — его индекс,
array — сам массив.

Например, мы хотим получить сумму всех элементов массива:

let arr = [1, 2, 3];

let result = arr.reduce(function(sum, item, index, array){

return sum + item

}, 0);

console.log(result);
